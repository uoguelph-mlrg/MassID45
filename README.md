# MassID45
This repo contains code for the MassID45 dataset: a multi-modal dataset for insect biodiversity with imagery and DNA at the trap and individual level.

# Installation
The CutLER, Mask R-CNN, Mask2Former, MaskDINO, and Grounded SAM 2 submodules each contain their own installation instructions. We recommend creating separate Python or Conda environments for each submodule.
